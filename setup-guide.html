Troubleshooting

If you're still getting "AI features not available" errors:

  • Check API Key: Make sure it's correctly pasted without extra spaces
  • Reload Extension: Go to chrome://extensions/ and click the reload button
  • Refresh Page: Reload the webpage you're testing on
  • Check Console: Press F12, go to Console tab, look for error messages
  • API Limits: Free tier has daily limits, check if you've exceeded them
